How do I add users to MicroMain Global?

If youâ€™re logged in as an administrator, you can add new users to the program. In the header menu or under settings, you will find Security. Under the security section, select Users, where you will find a list of all current users. To create a new user, click the green Create New Users button in the top right hand corner. You will complete the form, and click Save to save the new user to the program.

Users can be organized into Groups that determine what permissions the user has throughout the program. MicroMain Global provides templates for basic groups: Maintenance Admin, Maintenance Planners, and Maintenance Read-only. Admin users can create unique groups as well. This is done by selecting the Groups section of Security, and selecting from the green buttons in the top right hand corner: Create a new Group or Create from Template. When creating a Group from Template, you can change a couple things about the permissions of the preset groups. When creating a new Group, you will build the groupâ€™s permissions from scratch.


What is considered an Asset, and how do I manage them in MicroMain Global?

An asset is something of value that to be maintained. All of your companyâ€™s assets can be entered and tracked into Global. To do so, find the Asset section on the sidebar. Once there, you will see a list of all current assets. To add a new asset: click the green Create a New Asset button in the top right hand corner, complete the form, and click save. The asset will be added to list, and then its maintenance can be tracked.

MicroMain Global provides many ways to track maintenance on your assets, centered around Work Orders. Work Orders relate to unexpected maintenance. Users can report issues with assets (for instance, a light bulb goes out), which can then be turned into instructive work orders for laborers. Preventative Maintenance allows you to schedule work orders on assets that require cyclic maintenance. Think, for example, of oil changes on a car: you can schedule a work order every 3 months.


How do I create a Work Orders and what goes in one?

Every work order relates to at least one Asset and one Service performed on that asset. When completing the Work Order form, you should fill out as many categories as possible. This information will not only aid in the completion of the task, but also make reports more accurate. The following categories are included in every work order:

  • Site: The location of the Asset. You will select sites from a list of predetermined sites. Admin users can add or change sites. To enter the information about your sites, go to the Header Menu, and select Organization. There you can go to the Sites section, where there will be a list of current sites. To create a new site, click the green Create a New Site, complete the form, and click save.
  • Services: The service which needs to be performed on the asset. On the Work order, you will select from a list of possible services. You may select as many services as are required. Admin users can add or change services. To create a new service, an admin user should look to the administration section, under services, click the green button New Service, and complete the form.
  • Assets: Select the Asset or Assets that are being maintained.
  • Description: Here you can type the nature of the issue, including any special notes that the laborer may need when performing the task.
  • Location: Here you can type more information about the location of the Asset.
  • Priority: The priority relates to the urgency of the maintenance. Admin users can add or change priorities. In order to change or add priority levels: go to the Administration section, and select Priority. Once on the page, you will see a list of the current priorities. Click the green New Priority button in the top right hand corner, complete the form, and click save to add a custom priority.
  • Shop: The shop relates to the type of maintenance that is required. Here select a shop from a list of predetermined shops. Admin users can add or change shops. Look under Resources to find the shop page. There, you will see a list of shops. To add a shop, click the green New Shop button in the top right corner, complete the form, and click save. Issue By: Select a date when the Work Order will be issued. You can schedule work orders for the present day and into the future.
  • Start By: Select a date by which the maintenance should begin.
  • Due By: Select a date by which the maintenance should be completed.
